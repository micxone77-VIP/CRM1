⚙️ Google Sheets Connection

1

Publish your Google Sheet

In your Google Sheet → File → Share → Publish to web → Select "Entire Document" and "CSV" → Publish
Copy the URL for each tab you need.

2

Get your Sheet ID

From your Google Sheet URL:
https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit

3

Enter your Sheet ID below

The dashboard will load data from your published Google Sheet automatically.

🚀 Deploy to GitHub Pages

1

Upload to GitHub

Create a new repository → upload the index.html file → commit

2

Enable GitHub Pages

Repository Settings → Pages → Source: Deploy from branch → Branch: main → Save

3

Your live URL

Your dashboard will be live at:

https://YOUR-GITHUB-USERNAME.github.io/REPO-NAME/
4

Embed in Notion

In Notion → type /embed → paste your GitHub Pages URL → Done. The dashboard runs live inside Notion.
